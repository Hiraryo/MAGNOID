### 人間離れした新たな爽快アクションゲーム
人間界とMAGNOIDの世界をリンクさせる<br>
外で遊べない時はMAGNOIDで遊んで頂きたいので、 **現実とMAGNOIDの天候は反転させる**<br>
例）プレイヤーの世界の天候が雨なら、MAGNOID界の天候は晴れ<br>
例）プレイヤーの世界の天候が晴れなら、MAGNOID界の天候は雨<br>
天候（晴れ・雨・曇り）に応じたギミックを用意して、いつでもずっとMAGNOIDで楽しめるようにしたい。<br>
ゲームの開始時刻でMAGNOIDの環境を決定（朝・昼・夜）<br>

## インゲーム部分の抽出
- 主人公移動
- 主人公ジャンプ
- 主人公攻撃（通常攻撃と強攻撃）
- 主人公回避(入力方向へ素早く移動と操作反転攻撃のバリア)
- プレイヤーの腕前計算
- プレイヤーの腕前決定
- プレイヤーの腕前からゲームの難易度を決定
- プレイヤーの操作方法を記憶
- プレイヤーの行動パターンの解析
- ゲームオーバーの原因調査
- プレイヤーが住んでいる県を取得
- プレイヤーが住んでいる県から、気象庁のデータを参照して現実の世界の天候をゲームの世界に反映
- 敵の移動
- 敵の攻撃
- 主人公と敵のダメージ処理
- ゲームの難易度から敵の強さを決定
- ゲームの進行度に応じて
- 主人公によるギミック
  - 磁力で障害物を引き寄せる。中ボスの時にも使用可能。
  - ワープゾーンに入るとワープ（テレポートのような感じ）
  - 30秒間攻撃しなかったら体力アップ
  - 主人公が持つ4つの能力（磁力・AI(機械学習)・マインドリーディング・時間操作力）を駆使して、敵を倒しステージをクリアする
    - 磁力・・・地球自体が巨大な磁石のため、磁力操作により「電磁気バリア」を張り自分を防御。さらに障害物を引き寄せ、仕掛けを解く。（簡単で分かりやすい仕掛け。このゲームは謎解きがメインではない。）
    - AI(機械学習)・・・敵を倒して強くなる。（要するに主人公のレベルアップ）
    - マインドリーディング・・・ボスのみに有効でボスの攻撃パターンを読み取る（1戦に1回のみ）
    - 時間操作力・・・世界の時間を遅くし、大量の敵、又は連続で攻撃してくる敵を倒しやすくする
- 敵によるギミック
  - 操作反転攻撃・・・敵から操作反転攻撃を受けると、操作（移動方向）が反転する。20秒ほど操作が反転する
  - マイナス磁石・・・磁力をなくす事ができる。
- 天候によるギミック
  - 天候が雨になると主人公の移動速度が遅くなる
  - 各ステージに発電機が設置されており、発電機を壊そうとする敵が発電機を壊すと **画面が真っ暗になる** →一定時間が経過すると明るくなり発電機も復活する。発電機を壊そうとする敵を倒さない限り、何度も暗くなる。（画面上に発電機のHPを表示sる。発電機のHPが0になると暗くなる。 **プレイヤーは発電機を壊せない** ）
- 特殊道具によるギミック
  - その１（名称未決定）・・・（倒した敵が2体以上で敵同士マグネットでお互い引き寄せられる時に使用する。そして合成して敵のレア度に応じてS〜Dランクのソウルが貰える）
  - その２（名称未決定）・・・（これをはめると主人公の胸元にある丸いものが輝き、物凄い演算能力により全身が強化される。分身の場合これは何度も負けた敵に対して使うようにする。使用後の主人公は一旦オーバーヒートして動けなくなる）←最後の切り札的な特殊道具
  - その3(名称未決定)・・・（これを身につけて移動すると、自己回復能力が発動され少しづつ初期状態に近づいていく。だが回復効果はポーションの方が圧倒的に有利）
  - その4(名称未決定)・・・（主人公の分身を１体出現させ、この時の本人はN極、分身はS極となり発動中は2人の背中がくっつく状態になる。またこれを発動すればするほど分身も学習をし、賢くなれば分身自身で最適な攻撃方法を導き実行する）
- 上4つの特殊道具を駆使してゲームを攻略していくが、特殊道具の切り替えはハルがユーザーの操作を学習し、道具の切り替えはハルのAI機能で自動で行う
- 「移動」「回避」「ジャンプ」は必ずユーザーが行う。攻撃については基本ハルがオートで行う。（過去のプレイヤーの遊び方を参考にオートで戦う）オート中でプレイヤーは好きなタイミングで操作できる
- 障害物について
  - 電気爆弾・・・触れると感電する
