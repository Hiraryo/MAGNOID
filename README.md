### 人間離れした新たな爽快アクションRPGゲーム
人間界とMAGNOIDの世界をリンクさせる<br>
外で遊べない時はMAGNOIDで遊んで頂きたいので、 **現実とMAGNOIDの天候は反転させる**<br>
例）プレイヤーの世界の天候が雨なら、MAGNOID界の天候は晴れ<br>
例）プレイヤーの世界の天候が晴れなら、MAGNOID界の天候は雨<br>
天候（晴れ・雨・曇り）に応じたギミックを用意して、いつでもずっとMAGNOIDで楽しめるようにしたい。<br>
ゲームの開始時刻でMAGNOIDの環境を決定（朝・昼・夜）<br>

## ストーリーの概要（ざっくりと）
博士は、対TACの為に結成された精鋭部隊「〇〇」の特殊技術開発部の最高責任者である。<br>
特殊技術開発部とは、ロボットから生物に戻す為の薬品を研究・開発している。また薬品の研究と並行して「A」が率いるTACの反逆を阻止する為の、特殊技術の開発も行っている。<br>
またこの特殊技術を開発する為には、「IQ」という知能指数が高い秀才であり、「神7」という人間が生まれ持った能力が、全て研ぎ澄まされている必要がある。<br>
「神7」とは、五感・知覚・体感・官能・感受・感覚・精神のことである。<br>
<br>
**1.五感…視・聴・嗅・味・触の五つの感覚の総称。そして五感を研ぎ澄まして外界の状態や空間を把握する能力がある者<br>**
**2.知覚…感覚器官を通して外界の事物や身体内部の状態を認知する能力がある者<br>**
**3.体感…皮膚や内臓の諸器官に加えられた「暑さ」「寒さ」「痛み」「飢え」「渇き」「吐き気」など身体の異常状態にいち早く感じる能力がある者<br>**
**4.官能…五感によって、人の好みなど機械では扱えない感情が優れている。またその官能で物の品質の判定が出来る者<br>**
**5.感受…美醜やよしあし、相違などを感じとる心の働きのこと。ゆえにセンスがある者<br>**
**6.感覚…外界からの光・音・におい・味・寒温・触などの刺激を感じる働きと、それによって起こる意識。その意識を特殊技術に応用できる者<br>**
**7.精神…人間のこころ。またその知的な働きのことを表す。困難な状況下でも任務を遂行しようとする気力がある者<br>** 
<br>

博士はこの「神7」を元々持っており、機械工学を専門とし博士号も取得している。また博士は学生の頃からモノづくりが趣味であった。そして大学の院生時代「ロボットを作って自分の生活を豊かにしたい」と思い始める。その後、高い「IQ」を持ったエリートが斬新なロボットを開発している団体が存在することを知る。だが当時の博士の「IQ」ではこの団体と一緒にロボット開発が出来るほどの知能指数では無かった。<br>

そして、大学院を卒業する。それからは院生時代に思った事を叶える為とその団体に入りたいという強い意志で日々勉強に励む。<br>

だがある日、平凡な日常は突然失われた...<br>

博士が目指していた高い「IQ」を持った団体が生み出したロボット「A」が何らかの原因で勝手に暴走、複製、殺人を始めるようになってしまった。また「A」が複製したロボットの軍を「TAC」と呼ぶ。<br>

その頃博士は、60歳後半だがエリート達の「IQ」に相当する知能指数に達していた。その為「私ならTACという名の残虐的なロボットの暴走を阻止できる」と考え対TACとして精鋭部隊「〇〇」を立ち上げる。そして「HLW-001」を生み出す。<br>

だが、この頃はまだ博士のような「IQ」が高い人はごく少数であった。またこの時は「神7」の大切さには誰一人気付いていなかった。<br>

そのことから、当時の精鋭部隊では太刀打ちできず人類は滅亡の危機に迫っている。<br>

本作MAGNOIDは、この残虐的ロボット「TAC」と博士が最高責任者を務める「精鋭部隊」が生み出した人型ロボット「HLW-002」との戦いである。<br>


## ゲーム概要
- 人造人間の主人公HLW-002（愛称 ハル、ハルツー）が悪の組織「TAC(Trick Army Corps からくり兵団)」と戦い人類のピンチを救う爽快3DアクションRPGゲーム

## 主人公の名前「ハル」について
- 正式名称 **HLW-002**<br>[H = Human、 L = Last、　W = Weapons、 002 = 2号機]<br>(HLWとは高レベル廃棄物（原子力）という意味も持つ)
- 1号機の消息不明によって開発された人造人間2号機

## ゲーム全体の流れ
- ハルは、攻撃や回避、AI機能などを駆使しながら**森**、**都市**、**空中戦**を攻略した後にTACの核と戦い、ストーリー編終了。

## プレイスタイルについて
- 箱庭を繋ぎ合わせたオープンワールドです。
- FPS視点 / TPS視点を切り替えてワールドを攻略していきます。
- FPS視点にする事で、TPS視点では見えなかった所が見えたり、出来なかった事が出来るようになります。

## ワールド「森」について
- 神殿があり、苔に覆われている。
- 川があり、ジャングルクルーズのような感じの世界観
- 森の朝は明るく、涼しいため敵の数も少なく（他の敵は寝ている為）、攻撃力も通常より低く簡単
- 森の昼はとても明るいので、敵も活発になり、敵の数・攻撃力は通常で難易度も普通
- 森の夜は真っ暗なため夜行性の敵が多く、敵の数や攻撃力が上がり難易度も難しい
- ボスの天候は「雷」

## ワールド「都市」について
- 近未来な建物や物などをステージ内に配置する。
- 都市の朝は、敵の数も少なく、簡単
- 都市の昼は、敵の数・攻撃力・難易度とも普通
- 都市の夜は、敵の数・攻撃力・難易度とも難しい
※敵の数などの変動理由は**ワールド「森」**と同様（以降も同様）

## ワールド「空中戦」について
- 戦い方はシューティング
- イメージはスターフォックス
- 天候による難易度は、都市と同じ
- 初回プレイ時に操作のチュートリアルを入れる

## HLW-001戦について
- 2戦あり、
- 1戦目は、HLW-001戦の中ではもっとも簡単（しかし、今までのステージの昼以上の難易度）
- 1戦目はあまりギミックを出さない.（HLW-001がそもそも強い。）
- 2戦目はギミックも出していき、プレイスキルがまぁまぁ要求される難易度にする。
2戦目勝利後、ラスボス(TACの核)戦へ

## ラスボス（TACの核）戦について
- 3戦あり、
- 1戦目:ボスはバリアを使う事なく、常に攻撃してくるが攻撃力は弱く、隙が多い（連続攻撃はしない）ため倒しやすい。
- 2戦目:ボスはまだバリアを使わないが、攻撃力が上昇し、たまにクリティカルが出るようになる。そのため主人公はバリアを使うことが必須となる。
- 3戦目:ボスはバリアもたまに使うようになり、ダメージを与えにくくなる。さらに攻撃力も少し上昇し、移動速度も上昇し倒しにくくなる。またここで連続攻撃をするようになる。

## ラスボス（TACの核）の攻撃手段
- 通常攻撃（発動する時に合図がある）
  - ビーム(合図：後退する)→ハルはジャンプで回避
  - 地響き（合図：両腕をあげる）→ハルはジャンプで回避
- 特殊攻撃（低確率で発動、1戦目・2戦目・3戦目と進むに連れ発動確率が上がる）
  - 落雷（合図：画面が暗くなる）→ハルはバリアで回避
  - HP吸収（合図：画面がパッと光る）→パッと光った直後のハルの位置に向かって「物（詳細未決定）」を投げてくるので、ハルは回避をする。ハルに当たるとハルのHPが減る。
- 連続攻撃（3戦目のみ）
  - 電磁波（合図：目が光る）→ハルはバリアで回避（当たるとHPが結構ピンチに！）

## ハルのHP回復方法について
- 回復アイテムを消費してHPを回復
- ステージ内の所々に配置してある「回復ボックス」を壊すことでHPを回復

## ハルの異常状態と回復について
- 麻痺
- ハルは水溜りで動きが鈍くなる
- 異常状態は時間経過で自然回復する

## ステージのクリア / ゲームオーバー条件
- 雑魚敵を全て倒し、目的地に辿り着いた時をステージクリアとする
- ハルのHPが無くなった時をゲームオーバーとする→（[コンテニュー] or [諦める]かを選択してもらう）

  - HPを削りきる

  - ダッシュ（ハルに向かってくる）

  - ハルを見つけると向かってくる敵も用意
  - 監視カメラ

## 都市ボスへの特殊なダメージについて
- 両腕をはめている大砲（ドラえもんの空気砲のイメージ）を攻撃、破壊すると戦いが有利に
- 都市ボスの周りには3つのリザーバーがあり、それらを倒していくとさらに戦いが有利に
- リザーバーの蘇生は無く、リザーバーを倒すにはテクニックを必要とする。（例：ハルがリザーバーを攻撃し始めると、ボスの攻撃フローが変わりハルが受ける１回の被ダメが多くなる）

## 空中戦ボスへの特殊ダメージについて
- ボスが浮遊する為に必要なエンジン部分を攻撃すると通常より多くダメージを与えられる。
- エンジン部分は壊すことはできるが、何度も復元する。
- エンジン部分は攻撃しにくいようにする。（例：エンジン部分の周囲にバリア等を張り、近付きにくくする。）
- エンジン部分を破壊した回数によってボスの防御力が下がる
